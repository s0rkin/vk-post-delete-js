🔥 Рабочий скрипт для удаления ВСЕХ записей на личной странице!

📋 Инструкция по применению

Открой свою страницу ВКонтакте (https://vk.com/id...)
Прокрути стену вниз несколько раз, чтобы подгрузилось как можно больше старых записей. Чем больше прокрутишь — тем больше скрипт найдет.
Открой консоль браузера:

Chrome/Edge: Ctrl + Shift + J (Windows) или Cmd + Option + J (Mac)
Firefox: Ctrl + Shift + K (Windows) или Cmd + Option + K (Mac)
Вставь код в консоль и нажми Enter
Подтверди удаление в диалоговом окне
Не закрывай вкладку — скрипт будет удалять по одной записи в секунду и сам прокручивать страницу

⚠️ Важные моменты

Скорость: Скрипт намеренно медленный (1 запись в секунду), чтобы ВК не заблокировал тебя за подозрительную активность.
Блокировки: Если удалять слишком быстро, ВК может временно заблокировать возможность удалять записи. Если такое случилось — просто подожди 15-30 минут и запусти скрипт снова.
Остановка: Если хочешь прервать процесс, просто перезагрузи страницу.
Результат: Скрипт покажет в консоли лог удаления и в конце выдаст итоговое количество.
